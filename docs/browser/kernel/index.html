常见的浏览器内核有哪些？

对内核的了解程度

五大主流浏览器#

要想知道浏览器内核有哪些，首先要知道有哪些主流浏览器：

  • IE浏览器 windows系统自带浏览器
  • Safari Mac自带浏览器
  • Chrome 谷歌浏览器，当今市场的霸主
  • Firfox 火狐浏览器
  • Opera 欧朋浏览器

上述五个浏览器为目前主流的几个浏览器，关于国内其他的百度浏览器、360浏览器和手机端浏览器等等等等都是基于Webkit和Trident二次包装的。

四大内核#

Trident#

微软公司开发的内核，也可以称为IE内核，主要使用方是IE浏览器

Webkit#

苹果公司开发的内核，Webkit引擎包含了WebCode排版引擎和JavaScriptCode解析引擎 Safari和Chrome都使用此内核，Chrome后面版本的脚本解析使用自己研发的V8引擎。

Gcko#

Netscape公司Mozilla组织开发的内核，主要是Firfox浏览器使用

Blink#

Google和Opera Software开发的浏览器排版引擎 渲染引擎是开源引擎Webkit中WebCore组件的一个分支，并且在Chrome（28及往后版本）、Opera（15及往后版本）浏览器中使用
