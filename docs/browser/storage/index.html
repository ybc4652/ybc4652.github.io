浏览器存储的方式有哪些

对前端存储的理解和应用

cookie#

生命周期： 可以设置过期时间 存储大小： 4k

localStorage#

生命周期：永久，除非手动清理 存储大小：5M

sessionStorage#

生命周期：页面关闭就清理 存储大小：5M

indexDB#

生命周期：页面关闭就清理 存储大小：无限，存储于本地硬盘
